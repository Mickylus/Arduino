
# Relazioni di laboratorio

Questa repository raccoglie le relazioni e i sorgenti degli esperimenti con Arduino svolti durante le lezioni. Lo scopo è tenere traccia della documentazione, dei codici (file `.ino`) e dei materiali collegati ai progetti svolti in classe.

## Contenuto del repository
- `Arduino Day/`
	- `Umiditech.ino` — sketch usato per l'evento/esperimento "Arduino Day"
- `Progetto 3 protocolli/`
	- `RELAZIONE_PROGETTO.md` — relazione principale del progetto
	- `Arduino UNO/`
		- `uno_master_simplified.ino` — sketch semplificato per Arduino UNO (master)
	- `Arduino UNO R3/`
		- `uno_r3_slave_simplified.ino` — sketch semplificato per Arduino UNO R3 (slave)
	- `ESP32/`
		- `esp32_sender_simplified.ino` — sketch semplificato per ESP32 (sender)

## Come usare
- Aggiungi una breve relazione in formato Markdown per ogni esperimento (nella cartella del progetto corrispondente).
- Metti i file di codice (sketch `.ino`) nelle cartelle indicate sopra.
- Includi note, diagrammi, schemi dei collegamenti (breadboard/wiring) e riferimenti nelle relazioni.

Questo file fornisce un punto d'ingresso rapido per orientarsi nella repository.
